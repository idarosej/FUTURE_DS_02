# Customer Retention & Churn Analysis

## Objective

Analyze customer data to identify:

- Churn patterns
- Retention trends
- Customer lifetime value
- Key retention drivers

## Tools Used

- Python
- Pandas
- Matplotlib
- Excel
- Power BI

## Skills Gained

- Retention Analysis
- Cohort Analysis
- Customer Lifetime Metrics
- Insight-Driven Decision Making

## Deliverables

- Retention Dashboard
- Churn Analysis Report
- Actionable Recommendations

## Key Findings

- Customers with shorter subscriptions are more likely to churn.
- Long-term customers have higher lifetime value.
- Customer engagement improves retention.

## Recommendations

1. Improve customer onboarding.
2. Offer loyalty rewards.
3. Enhance customer support.
4. Identify at-risk customers early.
